Anish Ramasekar

Principal Software Engineer at Microsoft

I work on the Azure Container Upstream team building features for Kubernetes upstream and various CNCF projects that power Azure Kubernetes Service. I'm a co-chair of Kubernetes SIG Auth and a maintainer of the Secrets Store CSI Driver project.

I'm passionate about Kubernetes security, workload identity, and secrets management in cloud-native environments. My work spans across multiple domains including:

  • Kubernetes Security - Contributing to SIG-Auth and security-related features, container runtime security, workload identity patterns
  • Azure Workload Identity - Secrets management and secure authentication in cloud environments
  • Open Source - Active contribution to CNCF projects as a maintainer of several security-focused tools

I regularly speak at conferences and community events about Kubernetes security, sharing practical insights and real-world solutions. You can find my recent talks and presentations in the talks section.

I also write about Kubernetes security, cloud-native best practices, and emerging security technologies. Feel free to reach out if you'd like to discuss Kubernetes security, collaborate on open source projects, or have questions about cloud-native security best practices.
